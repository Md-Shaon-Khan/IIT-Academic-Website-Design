Habit Tracker

A Java-based solution to manage habits efficiently.

View on GitHub
Problem Statement

Tracking daily habits manually is difficult and inconsistent. People often forget to log habits, making it hard to maintain consistency and track progress. This system provides an easy digital way to manage daily habits.

Introduction

The Habit Tracker application allows users to record, update, and monitor their daily habits. It provides reminders, progress tracking, and insights to help users build and maintain good habits. Built using Java Swing for a simple interactive interface.

Features
  • Add, update, delete habits
  • Track daily, weekly, monthly progress
  • Set reminders and notifications
  • Visual representation of streaks and completion
Tools & Technology
  • Frontend: Java Swing
  • Backend: Core Java (OOP)
  • Database: MySQL
Impact

Habit Tracker helps users build better routines and improve productivity. It reduces forgetfulness, motivates streaks, and encourages long-term behavioral change. Digitizing habit tracking makes self-improvement measurable and engaging.
