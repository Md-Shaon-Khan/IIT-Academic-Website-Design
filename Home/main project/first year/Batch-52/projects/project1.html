Unique Question Generator

A teacher assistant tool for randomized, fair, and batch-specific question sets.

View on GitHub

Introduction

The Unique Question Generator is a smart tool built to assist teachers in preparing fair and unbiased exam question sets. It ensures randomized distribution of questions across batches, minimizing repetition and discouraging unfair exam practices.

Problem Statement

Traditional question paper setting often leads to repeated questions, lack of variation, and unfair practices among students. Teachers spend a significant amount of time preparing unique question sets for different batches, which is tedious and inefficient.

Features

  • Generates unique, randomized question sets.
  • Batch-specific customization.
  • Minimizes repetition across exams.
  • User-friendly interface for teachers.
  • Helps maintain fairness in exams.

Tools & Technology

This project was built using the following tools and technologies:

  • Frontend: HTML, CSS, Bootstrap
  • Backend: Java / Python
  • Database: MySQL / SQLite
  • IDE: VS Code / PyCharm / NetBeans
