Campus Navigator App

A mobile-first solution for navigating university campuses with real-time location tracking, building info, and event alerts.

View on GitHub

Introduction

The Campus Navigator App is designed to help students and visitors easily navigate university campuses. It provides real-time location tracking, building information, and event alerts, making campus life more connected and accessible.

Problem Statement

Navigating large campuses can be confusing, especially for new students and guests. Lack of centralized information about buildings and events leads to missed opportunities and wasted time. This app solves that by offering a smart, mobile-first navigation experience.

Features

  • Real-time campus navigation with GPS.
  • Building details and department info.
  • Event alerts and notifications.
  • Searchable campus map interface.
  • Mobile-first responsive design.

Tools & Technology

This project was built using the following tools and technologies:

  • Frontend: HTML, CSS, Bootstrap
  • Backend: JavaScript / Python
  • Database: Firebase / SQLite
  • IDE: VS Code / Android Studio

Impact

The Campus Navigator App enhances campus accessibility and engagement. It helps students find their way, stay informed about events, and connect with campus resources—all from their mobile devices.
